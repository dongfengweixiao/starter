# Changelog

## [1.0.0] - Initial Release

### Added
- Minimal Flutter MVVM starter template
- Cross-platform support (Linux, macOS, Windows, Android)
- MVVM architecture with get_it and watch_it
- Theme support (Light/Dark, Yaru/Phoenix)
- Localization ready
- Responsive design

---

This template was created by simplifying the [MusicPod](https://github.com/ubuntu-flutter-community/musicpod) project.
